# MapAR

Generated by **SKILL-IOS** (P7 #292) — first mobile-vertical skill
pack pilot. Validates the OmniSight P0-P6 mobile framework end-to-end.

## Build

```bash
# 1. Materialise the .xcodeproj (XcodeGen is the source of truth):
xcodegen generate

# 2. Resolve SPM dependencies:
xcodebuild -resolvePackageDependencies -scheme MapAR

# 3. Build & test in the iOS Simulator:
xcodebuild -scheme MapAR -destination 'platform=iOS Simulator,name=iPhone 15' test

# 4. Archive for App Store distribution (macOS host required):
bundle exec fastlane build
```

## Linux hosts

The scaffold itself renders on Linux, but iOS toolchains (`xcodebuild`,
`xcrun`, `gym`) are macOS-only. Use the OmniSight P1 (#286) delegation
matrix:

```bash
export OMNISIGHT_MACOS_BUILDER=cirrus-ci
bundle exec fastlane beta
```

## Distribute

| Lane                      | Track                                 |
|---------------------------|---------------------------------------|
| `fastlane beta`           | TestFlight (internal testers)         |
| `fastlane release`        | App Store Connect (review queue)      |

Distribution lanes call into `backend/app_store_connect.py` (P5 #290),
which gates submission behind the P6 (#291) ASC + Privacy compliance
bundle and the P3 (#288) codesign chain. The dual-sign +2 from
`backend/submit_rule.py` blocks production release until both an AI
reviewer (`merger-agent-bot`) and a human reviewer have signed off.
